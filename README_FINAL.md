## 📦 How to Use

### 1. Install Dependencies
```bash
cd AIAgent
npm install
```

### 2. Configure Environment
```bash
cp .env.example .env
# Add your API keys (Twilio, OpenAI, Deepgram, ElevenLabs)
```

### 3. Start Services
```bash
docker-compose up -d
```

### 4. Run Migrations
```bash
npm run db:migrate
```

### 5. Access Application
- **Frontend:** http://localhost:4000
- **API Gateway:** http://localhost:3000
- **Switch Language:** Click globe icon in sidebar
